Case Study #1

Work Organisation Toolset

⚠️ The Problem

BMW Plant Rosslyn's production teams managed rotation planning, attendance tracking, and skills matrices using paper forms and Excel. Team leaders spent 75% of their planning time on manual processes, training mismatches went undetected, and there was zero visibility into workforce readiness.

💡 The Solution

Built a complete workforce management platform with 7 live modules:

  • Rotation planning with skills highlighting
  • Attendance with escalated alerts (1hr → management at 4hrs)
  • Headcount management
  • Skills matrix (Levels 1-4 with 6-week decay logic)
  • Training documentation
  • Relief management
  • Auto end-of-shift reports

Case Study #2

Rework Tracking System

⚠️ The Problem

When vehicles entered rework bays, technicians had to manually look up defects in quality defect management system, cross-reference with their work area, and manually log time spent. This wasted time, caused errors, and made it impossible to measure rework efficiency.

💡 The Solution

Integrated vehicle positioning system (geofencing) (vehicle location) with quality defect management system (defect data) into a React app that:

  • Auto-detects vehicle entry/exit from rework bay
  • Pulls defects by VIN automatically
  • Filters defects by work area
  • Records session duration for each rework

🚀 Tech Stack

React.js vehicle positioning system Geofencing quality defect management system Defect API AWS Lambda DynamoDB

Case Study #3

AWS Serverless Data Pipeline (Hackathon)

⚠️ The Challenge

Build a complete end-to-end serverless data pipeline during BMW's AWS Hackathon - from raw data ingestion to an embedded analytics dashboard with authentication.

💡 The Solution

Designed and deployed a full pipeline:

  • S3 bucket for raw data landing
  • Lambda ETL for transformation
  • Glue crawlers for schema discovery
  • Athena for serverless querying
  • QuickSight dashboards
  • Cognito authentication
  • React embed for end users
